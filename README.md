# AI & ML Compendium for Power Systems

An authoritative compendium of Artificial Intelligence (AI) and Machine Learning (ML) resources for the Bulk Power System (BPS) and utility sector.

As a Professional Engineer (PE, Power) and Enterprise Architect, I found it difficult to find a centralized, high-quality list of modern AI/ML resources specific to the power systems domain. This repository is my effort to create that resource.

This list is intended for engineers, data scientists, and leaders working on grid modernization, renewable integration, load forecasting, and predictive maintenance.

Contributions are welcome.

## üèõÔ∏è Foundational Open Source Projects

This section highlights key open-source projects that form the backbone of modern AI-driven power system analysis.

### Core Projects
* **[PowerAgent](https://github.com/LF-Energy/power-agent)**: (LF Energy) LLM-powered tools and agentic AI for power system operations.
* **[GridFM](https://github.com/LF-Energy/gridfm)**: (LF Energy) Foundation Models for power grids.
* **[OpenSTEF](https://github.com/OpenSTEF/openstef)**: (LF Energy) Open-source Short-Term Energy Forecasting.
* **[PyPSA (Python for Power System Analysis)](https://github.com/PyPSA/pypsa)**: A toolbox for simulating and optimizing modern power systems.
* **[PyPSA-USA](https://github.com/PyPSA/pypsa-usa)**: An open-source power systems model of the bulk transmission system in the United States.
* **[HELICS (Hierarchical Engine for Large-scale Infrastructure Co-Simulation)](https://github.com/GMLC-TDC/HELICS)**: (GMLC) A co-simulation framework for energy systems.

### Resource Lists
* **[Awesome-AI-for-Electricity](https://github.com/AI4Electricity/Awesome-AI-for-Electricity)**: A curated list of AI/ML projects for the electricity sector.
